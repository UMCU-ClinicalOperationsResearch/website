# Team Member Photos

Add team member photos to this folder with the following naming convention:

- `kappen.jpg` - Dr. T.H. Kappen
- `van-loon.jpg` - Dr. L.M. van Loon
- `koomen.jpg` - Drs. E. Koomen
- `zoodsma.jpg` - Ruben Zoodsma
- `breteler.jpg` - Dr. Martine Breteler
- `nijman.jpg` - Dr. J. Nijman
- `boelhouwer.jpg` - Ileen Boelhouwer

Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

Recommended dimensions: 400x400 pixels (square format)
