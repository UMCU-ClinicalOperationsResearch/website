# Research Group Website - Project Setup

## Project Overview
Next.js website for research group with:
- Video hosting/embedding capabilities
- Scientific paper links and references
- Blog system with markdown support
- Team member bio pages

## Technology Stack
- Next.js 14+ with App Router
- TypeScript
- Tailwind CSS
- Markdown processing for blog posts
